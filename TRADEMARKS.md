# LEOS Project Name and Marks

The LEOS name, project logos, and related visual identifiers are administered
by Bad Tech Labs LLC separately from the Apache-2.0 license covering the source
code.

## Uses generally permitted

You may make accurate, non-misleading statements such as:

- “uses LEOS”;
- “compatible with LEOS”;
- “built for LEOS”;
- “forked from the LEOS project.”

You may also link to the project and reproduce unmodified project marks when
clearly identifying the official project.

## Uses requiring permission

Written permission is required for uses that could reasonably imply official
status, sponsorship, endorsement, certification, or control, including:

- naming a product, company, hosted service, or commercial distribution LEOS;
- using project marks as the primary branding of another offering;
- registering confusingly similar domains, social accounts, or application
  listings;
- materially modifying project logos for commercial branding.

## Forks and distributions

Forks may state their origin but should use a distinct name and branding when
changes could cause confusion about source, support, security, or release
status.

## No endorsement

Use of the open-source code does not imply endorsement, partnership,
certification, or support by the LEOS project or Bad Tech Labs LLC.

Questions about brand use should be directed through the contact method
published at `https://leosai.org`.
